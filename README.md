# Employee Tracker
A  CLI app to view and manage the departments, roles, and employees in a given company.

## Table of Contents

  * [Installation](#installation)
  * [Usage](#use)
  * [Questions](#questions)

## Installation

To install necessary dependencies, run the following command:

```
npm i
```
## use
```
npm run start
```
